To submit vehicle data, the following is required: pip3 install teslapy
Stop the Tornado Wallet, install teslapy and restart the wallet if you have not already done this.

The Tesla API uses Captchas to fetch an access token. The first time, the following will be displayed in the wallet terminal window: "Use browser to login. Page Not Found will be shown at success. Enter URL after authentication:" After entering your email address and Tesla account password, copy the auth.tesla.com URL into the wallet.py terminal window to fetch the access token.
